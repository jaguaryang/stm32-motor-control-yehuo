【*】 程序简介 
-工程名称：直流有刷电机-速度环电流环控制-位置式PID
-实验平台: 野火STM32 F407 开发板
-MDK版本：5.30
-HAL库版本：V1.7.6

【 ！】功能简介：
按下KEY1使能电机,并进入控制模式,按下KEY1\KEY2可以调整
占空比,以到达加减速的效果.

也可以通过上位机<野火多功能调试助手>----PID调试助手,查看现象.
在PID调试助手中,打开开发板对应的串口,单击下方启动即可.

【 ！】实验操作：

接线:
电机驱动板 5V\GND 					<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 				<----> 开发板 PA8\PA9
电机驱动板 SD 							<----> 开发板 PG12
电机驱动板 编码器输出A\B		<----> 开发板 PC6\PC7
电机驱动板 电压\电流检测 		<----> 开发板 PB0\PB1

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PA8\PA9上连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PA8\PA9

电机使能引脚:
电机驱动板 SD <----> 开发板 PG12
	
编码器信号:
用于对电机进行测速
电机驱动板 编码器输出A\B		<----> 开发板 PC6\PC7

电压电流采集:
电机驱动板 信号检测 电压\电流 		<----> 开发板 PB0\PB1
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 168MHz
-APB2 = 84MHz 
-APB1 = 42MHz

C.浮点运算单元：
  不使用
