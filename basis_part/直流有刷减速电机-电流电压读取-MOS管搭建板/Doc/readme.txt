【*】 程序简介 
-工程名称：直流有刷减速电机-电流电压读取-MOS管搭建板
-实验平台: 野火STM32 F407 开发板
-MDK版本：5.30
-HAL库版本：V1.7.6

【 ！】功能简介：
按下KEY1使能电机,并进入控制模式,
按下KEY2关闭电机。
按下KEY3\KEY4可以调整占空比,以到达加减速的效果.
串口会打印采集得到的电压电流信息.

【 ！】实验操作：

接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PA8\PA9
电机驱动板 SD 			<----> 开发板 PG12
电机驱动板 电压\电流	<----> 开发板 PB0\PB1

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PA8\PA9上连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PA8\PA9

电机使能引脚:
电机驱动板 SD <----> 开发板 PG12

电压电流采集引脚:
电机驱动板 电压	<----> 开发板 PB0
电机驱动板 电流	<----> 开发板 PB1
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 168MHz
-APB2 = 84MHz 
-APB1 = 42MHz

  不使用
