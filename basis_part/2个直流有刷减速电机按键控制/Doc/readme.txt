【*】 程序简介 
-工程名称：2个直流有刷减速电机按键控制
-实验平台: 野火STM32 F407 开发板
-MDK版本：5.30
-HAL库版本：V1.7.6

【 ！】功能简介：
上电即使能电机,并进入控制模式,按下KEY1\KEY2可以调整
占空比,以到达加减速的效果.

【 ！】实验操作：

电机1接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PA8\PA9
电机驱动板 SD 				<----> 开发板 PG12

电机2接线:
电机驱动板 5V\GND 		<----> 开发板 5V\GND
电机驱动板 PWM1\PWM2 	<----> 开发板 PI5\PI6
电机驱动板 SD 				<----> 开发板 PE6

下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
两路PWM输出到PA8\PA9上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PA8\PA9

电机使能引脚:
电机驱动板 SD <----> 开发板 PG12

定时器TIM8输出PWM信号:
两路PWM输出到PI5\PI6上,连接到驱动板上的PWM信号输入.
电机驱动板 PWM1\PWM2 <----> 开发板 PI5\PI6

电机使能引脚:
电机驱动板 SD <----> 开发板 PE6
